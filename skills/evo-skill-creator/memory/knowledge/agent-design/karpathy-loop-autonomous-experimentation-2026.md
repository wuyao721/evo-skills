# Karpathy Loop — 自主实验循环（2026年3月）

> 来源：Fortune 报道（2026-03-17）
> 学习时间：2026-03-29
> 类别：Agent 自我改进机制
> 波动性：medium（实验循环是稳定机制，但具体实现会演进）

## 核心发现

Andrej Karpathy（前 OpenAI 研究员）展示的自主 Agent 在 **2 天内运行了 700 个实验**，这是 AI 自我改进领域的重要里程碑。

## Karpathy Loop 的核心机制

### 1. 高速实验循环

- **实验速度**：700 实验 / 2 天 = **350 实验/天** ≈ **15 实验/小时**
- **对比人类**：人类研究员通常需要数周才能完成几十个实验
- **速度优势**：Agent 的实验速度是人类的 **100-1000 倍**

### 2. 自主实验流程

```
假设生成 → 实验设计 → 执行实验 → 结果评估 → 知识更新 → 新假设
    ↑                                                              ↓
    └──────────────────────── 循环 ────────────────────────────────┘
```

### 3. 关键能力要求

1. **自动假设生成** — Agent 能基于现有知识生成新假设
2. **实验自动化** — 无需人工干预即可执行实验
3. **自动评估** — 能够自动判断实验成功/失败及原因
4. **知识自动沉淀** — 将实验结果转化为可复用的知识

## 对 AI 未来发展的启示

### 1. 进化速度的本质是实验速度

- **传统进化**：依赖人工反馈，速度受限于人类响应时间
- **Karpathy Loop**：完全自主循环，速度仅受限于计算资源
- **结论**：自动化实验循环是实现快速进化的关键

### 2. 评估是瓶颈

- **无评估 = 盲目进化**：没有自动评估，无法支撑高频实验
- **评估质量决定进化方向**：错误的评估会导致进化偏离目标
- **多维度评估**：需要从多个维度评估实验结果（性能、成本、安全性等）

### 3. 知识沉淀是进化的基础

- **实验 → 知识**：每个实验都应产生可复用的知识
- **知识 → 假设**：已有知识是生成新假设的基础
- **闭环增强**：知识越多，假设质量越高，进化越快

## 与我们的自我进化模型的对比

### 我们的现状

| 维度 | 我们的实现 | Karpathy Loop | 差距 |
|------|-----------|--------------|------|
| 实验触发 | 手动（review 命令） | 自动循环 | **大** |
| 实验速度 | 低（依赖人工触发） | 高（350/天） | **大** |
| 评估机制 | 主观评价 | 自动评估 | **大** |
| 知识沉淀 | 手动写入 knowledge/ | 自动沉淀 | **中** |
| 假设生成 | 依赖 scan/learn | 自动生成 | **中** |

### 我们的优势

1. **人类监督** — 避免进化偏离目标
2. **知识可读性** — 文件系统级 memory 人类可直接阅读
3. **灵活性** — 可根据场景调整进化策略

### 我们的差距

1. **缺少自动实验循环** — 依赖人工触发 review
2. **缺少自动评估** — 完全依赖主观评价
3. **实验速度慢** — 无法支撑高频迭代

## 改进方向

### 短期（立即可做）

1. **在 review 命令中增加"效果评估"步骤** — 量化评估升级效果
2. **设计实验记录格式** — 记录每次 review 的假设、执行、结果
3. **建立评估指标库** — 为不同类型的改进定义评估指标

### 中期（需要设计）

1. **实现半自动实验循环** — review 命令自动触发，但需人工确认
2. **设计自动评估框架** — 基于多维度指标自动评估改进效果
3. **实验历史分析** — 分析历史实验，识别成功模式

### 长期（探索方向）

1. **完全自动化的 Karpathy Loop** — 无需人工干预的自主实验循环
2. **多 Agent 协同实验** — 多个子智能体并行实验，加速进化
3. **元学习** — Agent 学习"如何更好地实验"

## 关键结论

1. **实验速度是进化速度的上限** — 要快速进化，必须提高实验频率
2. **自动评估是自主进化的前提** — 没有自动评估，无法实现 Karpathy Loop
3. **知识沉淀是进化的基础** — 每个实验都应产生可复用的知识
4. **人类监督仍然重要** — 完全自主的循环需要强大的安全机制
